STRIGOI BF6 — Descarga

Este es el paquete oficial STRIGOI BF6. Incluye la app local (Matrix server), HUD y lanzadores para macOS/Windows.

Descargar paquete (ZIP) Instrucciones
Nota: Para que el botón de descarga funcione en tu web pública, sube STRIGOI_BF6_APP.zip al mismo sitio (o cambia el enlace al archivo público en GitHub Releases / S3 / Dropbox).

Instalación rápida

  1. Descarga y descomprime el ZIP.
  2. En macOS: da permisos al lanzador: chmod +x Iniciar_STRIGOI.command
  3. Haz doble clic en Iniciar_STRIGOI.command para abrir la Terminal y lanzar la app.
  4. Abre strigoi_hud_matrix_v5.html en el navegador y activa Auto‑sync.

Para usuarios avanzados: puedes ejecutar directamente en Terminal: python3 bf6_local_server_input_mac_v5_2_1.py

Cómo publicar esto (sugerencias rápidas)

  • GitHub Releases: sube el ZIP como release — recomendado. URL del ZIP en el botón.
  • GitHub Pages / Netlify / Vercel: sirve la página estática. Si usas GitHub Pages, añade el ZIP al repo o enlaza al Release.
  • Dropbox / Google Drive: comparte enlace directo (asegúrate de usar link directo al archivo).

Si quieres, te preparo el repositorio listo para subir a GitHub con el index.html ya configurado.
